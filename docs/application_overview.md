# Application Overview

## Application Name

Explore Salzburg: Interactive WebGIS Application for Urban Amenities

## User Groups

The application is suitable for:

- students learning geovisualization and WebGIS;
- residents exploring public amenities;
- visitors looking for cultural and public service locations;
- portfolio reviewers evaluating practical geoinformatics skills.

## Core Workflow

1. Open the application.
2. Use category filters to control visible amenity types.
3. Search for a place or attribute.
4. Click a marker to inspect details.
5. Use click analysis to assess amenities around a selected location.
6. Export or print the resulting map view.

## Design Principles

- Keep the basemap visually quiet so thematic points remain dominant.
- Use strong but consistent colours for amenity categories.
- Place core controls in the left sidebar.
- Place interpretive outputs in the right insight panel.
- Keep the map as the main visual focus.
- Provide immediate feedback when the user filters, selects or analyses data.
